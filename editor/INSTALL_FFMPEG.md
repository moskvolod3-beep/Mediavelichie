# Установка FFmpeg

FFmpeg необходим для работы приложения. Вот несколько способов установки:

## Способ 1: Автоматическая установка (Windows)

1. Запустите `install_ffmpeg.bat` **от имени администратора** (правой кнопкой → "Запуск от имени администратора")
2. Дождитесь завершения установки
3. **Перезагрузите компьютер** или перезапустите командную строку

## Способ 2: Ручная установка (Windows)

1. Скачайте FFmpeg:
   - Перейдите на https://www.gyan.dev/ffmpeg/builds/
   - Скачайте `ffmpeg-release-essentials.zip`

2. Распакуйте архив:
   - Распакуйте в `C:\ffmpeg`

3. Добавьте в PATH:
   - Нажмите Win + Pause/Break
   - Выберите "Дополнительные параметры системы"
   - Нажмите "Переменные среды"
   - В "Системные переменные" найдите `Path` и нажмите "Изменить"
   - Нажмите "Создать" и добавьте: `C:\ffmpeg\bin`
   - Нажмите "ОК" во всех окнах

4. Перезагрузите компьютер

5. Проверьте установку:
   ```cmd
   ffmpeg -version
   ```

## Способ 3: Через Chocolatey (Windows)

Если у вас установлен Chocolatey:
```cmd
choco install ffmpeg
```

## Способ 4: Через Winget (Windows 10/11)

```cmd
winget install ffmpeg
```

## Linux

```bash
sudo apt-get update
sudo apt-get install ffmpeg
```

## macOS

```bash
brew install ffmpeg
```

## Проверка установки

После установки откройте командную строку и выполните:
```cmd
ffmpeg -version
```

Если вы видите информацию о версии FFmpeg, значит установка прошла успешно!
