# Установка Python

Python необходим для работы приложения. Вот как его установить:

## Windows

### Способ 1: Официальный сайт (рекомендуется)

1. **Скачайте Python:**
   - Перейдите на https://www.python.org/downloads/
   - Нажмите кнопку "Download Python" (скачается последняя версия)

2. **Установите Python:**
   - Запустите скачанный файл
   - **ВАЖНО**: Отметьте галочку **"Add Python to PATH"** внизу окна установки
   - Нажмите "Install Now"
   - Дождитесь завершения установки

3. **Проверьте установку:**
   - Откройте командную строку (Win+R, введите `cmd`, Enter)
   - Введите: `python --version`
   - Должна появиться версия Python (например, Python 3.11.0)

4. **Если Python не найден:**
   - Перезагрузите компьютер
   - Или добавьте Python в PATH вручную (см. ниже)

### Способ 2: Microsoft Store

1. Откройте Microsoft Store
2. Найдите "Python 3.11" или "Python 3.12"
3. Нажмите "Установить"
4. После установки Python будет автоматически добавлен в PATH

### Способ 3: Через Winget (Windows 10/11)

Откройте PowerShell или командную строку и выполните:
```cmd
winget install Python.Python.3.11
```

### Ручное добавление Python в PATH

Если Python установлен, но не найден:

1. Найдите, где установлен Python (обычно `C:\Users\ВашеИмя\AppData\Local\Programs\Python\Python3XX` или `C:\Python3XX`)
2. Нажмите Win + Pause/Break
3. Выберите "Дополнительные параметры системы"
4. Нажмите "Переменные среды"
5. В "Системные переменные" найдите `Path` и нажмите "Изменить"
6. Нажмите "Создать" и добавьте пути:
   - `C:\Users\ВашеИмя\AppData\Local\Programs\Python\Python3XX`
   - `C:\Users\ВашеИмя\AppData\Local\Programs\Python\Python3XX\Scripts`
7. Нажмите "ОК" во всех окнах
8. Перезагрузите компьютер

## Проверка установки

После установки откройте командную строку и выполните:
```cmd
python --version
```

Если вы видите версию Python, значит установка прошла успешно!

## Альтернативные команды

Если `python` не работает, попробуйте:
- `py --version` (лаунчер Python на Windows)
- `python3 --version`
