# Медиа Величия

Сайт агентства видеомаркетинга полного цикла.

## Структура проекта

- `index.html` - Главная страница
- `services.html` - Страница услуг
- `portfolio.html` - Портфолио
- `brandup.html` - Страница брендапа
- `project.html` - Страница проекта
- `contacts.html` - Контакты
- `privacy.html` - Политика конфиденциальности
- `404.html` - Страница ошибки 404

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript
- SVG иконки
- PNG изображения

## Запуск

Просто откройте `index.html` в браузере или используйте локальный сервер.
