## Следующие шаги

### 1. Настройка пользователя Git (если еще не настроено)

```bash
git config --global user.name "Ваше Имя"
git config --global user.email "ваш@email.com"
```

Или только для этого проекта:

```bash
git config user.name "Ваше Имя"
git config user.email "ваш@email.com"
```

### 2. Создание первого коммита

```bash
git commit -m "Initial commit: проект Медиа Величия"
```

### 3. Настройка удаленного репозитория (опционально)

Если у вас есть репозиторий на GitHub/GitLab/Bitbucket:

```bash
# Добавить удаленный репозиторий
git remote add origin https://github.com/ваш-username/ваш-репозиторий.git

# Переименовать ветку в main (если нужно)
git branch -M main

# Отправить код на удаленный репозиторий
git push -u origin main
```

### 4. Основные команды Git

```bash
# Проверить статус
git status

# Добавить все изменения
git add .

# Добавить конкретный файл
git add имя-файла

# Создать коммит
git commit -m "Описание изменений"

# Посмотреть историю коммитов
git log

# Отправить изменения на удаленный репозиторий
git push

# Получить изменения с удаленного репозитория
git pull
```
