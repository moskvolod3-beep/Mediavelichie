## ТЕХНИЧЕСКИЕ ОСОБЕННОСТИ

### Сложные элементы

1. **Декоративный фон процесса (Component3):**
   - Множественные наложенные элементы
   - Эффект `mix-blend-exclusion`
   - Требует точного позиционирования

2. **Карусель отзывов:**
   - Перекрывающиеся элементы
   - Поворотные элементы (180deg)
   - Требует JavaScript для функциональности

3. **Варианты карточек направлений:**
   - Два варианта отображения (Default/Variant2)
   - Разное позиционирование overlay
   - Возможно, переключение при hover

4. **Сложная сетка услуг:**
   - Разные размеры карточек
   - Асимметричная компоновка
   - Требует точного Grid/Flexbox

5. **Hero секция с боковой панелью:**
   - Полупрозрачный фон с blur
   - Наложение на фоновое изображение
   - Фиксированная навигация

### Особенности реализации

1. **Backdrop-filter:**
   - Использование `backdrop-blur-[5px]` для overlay
   - Может требовать fallback для старых браузеров

2. **Градиенты:**
   - Radial gradients для кнопок и карточек
   - SVG градиенты в некоторых элементах

3. **Позиционирование:**
   - Много абсолютного позиционирования
   - Использование calc() для точных позиций
   - Процентные значения для адаптивности

4. **Типографика:**
   - Очень большие размеры шрифтов (160px, 260px)
   - Требует оптимизации для производительности
   - Возможно, использование `text-shadow` для эффектов
